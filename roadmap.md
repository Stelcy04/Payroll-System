# Roadmap

## Etapa 1: base operativa

- empleados
- periodos
- salary advances
- planillas basicas
- reportes PDF y Excel

## Etapa 2: experiencia de usuario

- filtros por estado
- busqueda por nombre, codigo o referencia
- mejores mensajes de error
- acciones visuales mas claras
- consistencia de subpantallas en todos los modulos

## Etapa 3: motor automatico de planilla

- asistencia
- horas extra
- ausencias
- bonos
- deducciones configurables
- prestamos
- reglas automaticas por periodo

## Etapa 4: negocio real

- autenticacion
- multiempresa
- auditoria de acciones
- aprobacion y cierre formal
- historico por periodo

## Etapa 5: produccion

- migracion de JSON a base de datos relacional
- roles y permisos
- copias de seguridad
- despliegue
- monitoreo
